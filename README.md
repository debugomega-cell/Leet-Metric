LeetMetric
This project fetches and displays basic LeetCode user statistics.

The user enters a LeetCode username, and the app retrieves data from an external API and shows:
Easy, Medium, and Hard problems solved
User ranking
Acceptance rate
Contribution points

API Used
This project uses the following public API:
https://leetcode-stats-api.herokuapp.com/{username}

vercel link :https://leetmatric-five.vercel.app/
